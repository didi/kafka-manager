# 升级至`2.5.0`版本

`v2.5.0` 主要是修改了产品的名字及相关的模块名，在整个的代码层面修改的不多，使用2.5.0时，直接替换原有的安装包后进行启动即可。

新旧模块名的对应关系：
- kafka-manager-dao 变更为 logikm-persistence
- kafka-manager-web 变更为 logikm-rest
